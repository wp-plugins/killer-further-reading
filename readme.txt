Killer further reading recommends content to user based on a tag and excludes already visited content.

== Description ==

There is no plugin that uses tags for showing further reading and then this plugin recommends further reading only those content that user has not seen. The tracking is done through cookie being placed for the content. 

== Installation ==

Please Follow the exact Steps to install and activate this plugin.

   1. Upload the plugin folder killer-further-reading at wp-content/plugins.
   2. Activate the plugin through the 'Plugins' menu in WordPress
   3. Drag Killer Further Reading to the widget area for options to choose the tag for showing content.

== Frequently Asked Questions ==

= Installed and activated the Plugin but cannot find any option for the plugin? =

This is a widget and so you will see a new widget Killer Further Reading widget that can be dragged anywhere you prefer on your sidebar, or in footer.
